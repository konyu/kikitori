# 12 — i18n（国際化対応）

## 目的
UI 文字列・ログ出力を日本語と英語で切り替え可能にする。OS の言語設定に応じて自動判定。

## Python 版参照
`kikitori/i18n.py`:
```python
STRINGS: dict[str, dict[str, str]] = {
    "ja": {
        "app.tooltip": "Kikitori",
        "app.status.idle": "○ 待機中",
        "app.status.recording": "🔴 録音中...",
        # ... 70+ キー ...
    },
    "en": {
        "app.tooltip": "Kikitori",
        "app.status.idle": "○ Idle",
        # ...
    }
}

def t(key: str, lang: str | None = None) -> str:
    if lang is None: lang = "ja"
    if lang not in STRINGS: lang = "ja"
    return STRINGS[lang].get(key, STRINGS["ja"].get(key, key))
```

### 未定義キーのフォールバック
1. 指定言語の辞書 → なければ日本語辞書 → なければキー名そのまま

## Swift 実装仕様

### TranslationKey enum
```swift
public enum TranslationKey: String, CaseIterable {
    case menuSettings
    case menuQuit
    case statusIdle
    case statusRecording
    // ...
}
```

### I18n クラス
```swift
public final class I18n: Sendable {
    public private(set) var language: String  // "ja" or "en"
    
    public init(language: String? = nil)  // nil → OS から自動検出
    public func t(_ key: TranslationKey) -> String
    
    public static func osLanguage(preferredLanguages: [String]? = nil) -> String
    // Locale.preferredLanguages の先頭が "ja" で始まれば "ja"、それ以外 "en"
}
```

### 文字列辞書
```swift
private let strings: [String: [TranslationKey: String]] = [
    "ja": [
        .menuSettings: "設定",
        .menuQuit: "終了",
        // ...
    ],
    "en": [
        .menuSettings: "Settings",
        .menuQuit: "Quit",
        // ...
    ]
]
```

### 自動言語検出
```swift
public static func osLanguage(preferredLanguages: [String]? = nil) -> String {
    let langs = preferredLanguages ?? Locale.preferredLanguages
    guard let first = langs.first else { return "en" }
    return first.hasPrefix("ja") ? "ja" : "en"
}
```

### 使用例
```swift
// AppDelegate
let lang = I18n.osLanguage()
let i18n = I18n(language: lang)
// menu.addItem(NSMenuItem(title: i18n.t(.menuQuit), ...))
```

### 設定ファイル連携
```yaml
ui_language: ja  # "ja" または "en"。未指定時は OS 自動検出
```

### 翻訳対象
| キー | 日本語 | English |
|-----|-------|---------|
| menuSettings | 設定 | Settings |
| menuQuit | 終了 | Quit |
| statusIdle | 待機中 | Idle |
| statusRecording | 録音中 | Recording |
| settingsTitle | Kikitori 設定 | Kikitori Settings |
| ... | ... | ... |
（全キーは実装時に Python 版 i18n.py から全コピー）

### 単体テスト
- `osLanguage()`: "ja" プレフィックスで "ja"、"en" で "en"
- 未定義キーでフォールバック
- 全キーが両言語で定義済みか検証
