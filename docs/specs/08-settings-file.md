# Settings File

YAML 形式の設定ファイルを `~/.kikitori_settings.yaml` に保存・読み込み。

## 設定項目
- `language`: 認識言語（"ja", "en" 等）
- `hotkey`: ホットキー設定（デフォルト "option"）
- `min_duration_ms`: 最低録音時間（デフォルト 500）
- `max_duration_sec`: 最大録音時間（デフォルト 60）
- `silence_rms_threshold`: 無音判定閾値（デフォルト 0.0001）
- `debug`: デバッグモード有効/無効
- `ui_language`: UI 言語（"ja" または "en"）
