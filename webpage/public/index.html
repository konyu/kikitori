macOS ネイティブアプリ

話すだけ、

Kikitori はホットキー押下中にマイク入力を録音し、解放時に音声をテキストに変換して自動ペースト。ネットワーク不要のオンデバイス音声認識で、軽量かつ高速に動作します。

Features

軽量・高速な音声入力体験

Apple の最新 Speech Framework を活用し、macOS に自然に馴染む設計で作られたネイティブアプリです。

Swift / SwiftUI ネイティブ実装

Electron フレームワークを使わない純粋な macOS アプリ。メモリ使用量を抑え、素早く起動します。

オンデバイス音声認識

SpeechAnalyzer / SpeechTranscriber を利用し、音声データを外部に送信せずにローカルで認識。オフラインでも使えます。

Liquid Glass オーバーレイ UI

録音中は画面下部に「Apple Liquid Glass」調のリアルタイム波形と鳥アイコンが表示。今録音されているかが直感的にわかります。

GUI で簡単カスタマイズ

認識言語、UI 表示言語、ホットキー、録音時間、無音判定しきい値などを設定画面から直感的に変更できます。

誤変換の自動修正

Corrections 機能で「use effect → useEffect」など、よく間違える単語や専門用語を自動で置換できます。

自動アップデート

Sparkle フレームワークを組み込み、新しいバージョンがリリースされたらメニューから簡単に更新できます。

How it works

3 ステップで使い始められます

複雑な操作は不要。ホットキーを押す・話す・離すだけで、今カーソルがある場所に文字が入力されます。

ホットキーを押し続ける

デフォルトでは Fn（🌐）キーを押している間、録音が開始されます。

Fn Hold

話す

画面下部にガラス調のオーバーレイが表示され、声の大きさに合わせて波形がリアルタイムに動きます。

キーを離して自動ペースト

ホットキーを離すと録音が終了し、認識結果をクリップボード経由でカーソル先に自動入力します。

Install

インストール

Releases ページから最新の DMG をダウンロードし、Applications フォルダへ入れるだけです。

インストール手順

  1. Releases ページから最新の Kikitori-x.x.x.dmg をダウンロードしてください。
  2. ダウンロードした DMG ファイルを開きます。
  3. 中にある Kikitori.app を Applications フォルダへドラッグ＆ドロップします。
  4. Applications フォルダから Kikitori を起動してください。

初回起動時の「開発元が未確認」警告

「開発元を確認できない」「マルウェアがないか検証できません」と表示された場合、以下のいずれかで許可してください。

  1. A. システム設定から許可: 警告ダイアログを「完了」で閉じ、システム設定 → プライバシーとセキュリティ → 「このまま開く」をクリックします。
  2. B. ターミナルで検疫を解除: 以下のコマンドを実行します。
xattr -rd com.apple.quarantine /Applications/Kikitori.app

System Requirements

動作環境と必要な権限

Apple Silicon Mac と macOS 26.0 以上が必要です。初回起動時に 3 つの権限を許可してください。

マイク

音声入力をキャプチャするために使用します。
プライバシーとセキュリティ → マイク

音声認識

Speech Framework による音声→テキスト変換に使用します。
プライバシーとセキュリティ → 音声認識

アクセシビリティ

ホットキー監視と自動ペースト（Cmd+V エミュレーション）に使用します。
プライバシーとセキュリティ → アクセシビリティ

Corrections

専門用語も正しく入力

音声認識で決まって間違われる言い回しや技術用語は、校正設定に登録して自動置換できます。YAML ファイルを直接編集することも可能です。

  • 追加 / 編集 / 削除が GUI で直感的に操作可能
  • ~/.kikitori/corrections.yaml として保存
  • 例: "use effect" → "useEffect"
corrections.yaml
# よく間違える単語を置換
corrections:
  - wrong: "use effect"
    right: "useEffect"
  - wrong: "じゃんご"
    right: "Django"
  - wrong: "デーブオプス"
    right: "DevOps"
